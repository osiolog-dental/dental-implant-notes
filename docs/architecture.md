# Architecture

DentalHub is a full-stack dental implant case management system. The frontend is built with React 19 and packaged as native iOS and Android apps via Capacitor, allowing the web codebase to ship as a true mobile app without a rewrite. The backend is a FastAPI application backed by a PostgreSQL database (AWS Lightsail Managed PostgreSQL), using SQLAlchemy for ORM and Alembic for schema migrations. Authentication is handled by Firebase Authentication — the frontend exchanges credentials with Firebase and passes a verified ID token to the backend on every request. Clinical images (radiographs, photos) are stored in a private AWS S3 bucket in the `ap-south-1` (Mumbai) region; the backend issues presigned URLs for direct browser-to-S3 uploads and secure downloads, so image bytes never pass through the app server. The app server itself runs on AWS Lightsail in the same Mumbai region. Push notifications for osseointegration milestones are delivered via Firebase Cloud Messaging (FCM).
